# CaoxWear API

Backend Spring Boot para el proyecto final de e-commerce de ropa.

## Ejecutar

```bash
set "JAVA_HOME=C:\Program Files\Android\Android Studio\jbr"
mvnw.cmd spring-boot:run
```
